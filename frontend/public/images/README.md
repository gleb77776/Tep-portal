# Фотографии сотрудников

Скопируйте сюда фото из телефонного справочника:

- Источник: папка `static/images` проекта телефонного справочника (на сервере tepmapp25 — `C:/sites/phonebook/static/images` или аналогичная)
- Или скачайте с http://tepmapp25:5000/images/ при наличии доступа

**Формат имён:** `BatyanovskiyGV.png`, `SharovIA.png` (Фамилия + инициалы в PascalCase, .png)

Если файла нет — показываются инициалы пользователя.
